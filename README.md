# Excel Sales Dashboard 
An interactive sales dashboard built using Microsoft Excel to analyze sales performance across products, locations, and time.

## Features
- Dynamic dashboard using Pivot Tables
- Interactive slicers (Date, Place, Products)
- KPI cards for Total Sales & Total Amount
- Monthly, Product-wise, and Location-wise analysis
- Clean and professional dashboard design

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Basic Excel formulas

## Dataset Columns
- Date
- Salesperson
- Product
- Place
- Price
- Units
- Amount
